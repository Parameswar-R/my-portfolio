EOS (Energy Operation System)

EOS is a comprehensive, cloud-native web application designed to streamline energy operations, providing a user-friendly interface for managing customer enrollments, tracking energy consumption, generating invoices, and processing payments. Built with microservices architecture, it ensures efficiency and scalability.

Technologies Used:
C# ASP.NET Core Web API Angular SQL Server Azure Azure ServiceBus GrayLog
Key Features
  • Plan Module: Manages various customer plans, pricing strategies, and configurations.
  • Enrollment Module: Handles customer enrollments with automated workflows.
  • Invoice Module: Generates usage-based invoices and enables payment tracking.
  • Payment Module: Manages secure payment processing and automated deductions.
